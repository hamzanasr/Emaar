# إعمار (Emaar) — Construction Platform

> المنصة الموحدة لقطاع المقاولات والإنشاءات والتشطيب

## الهوية البصرية

- **التصميم**: فاخر، سينمائي، مينيمالي
- **الألوان**: أزرق غامق + ذهبي + أحمر كرمسون
- **الخط**: Cairo (Google Fonts)
- **التقنيات**: Next.js 14, Tailwind CSS, Zustand, React Query

## Quick Start

```bash
npm install
npm run dev
```

## الروابط

- **الواجهة الأمامية**: http://localhost:3000

## النسخة التجريبية

- رمز التحقق الثابت: `1234`
- أي رقم هاتف يعمل
